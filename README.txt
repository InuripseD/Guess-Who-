/!\Utilise JAVA 11 ou une version supperieur/!\

1-Telecharger le code source.
2-Extraire le dossier qui-est-ce-main du .zip.
3-Se rendre dans le dossier qui-est-ce-main.
4-Executer le fichier prerequisLinux.bat sous linux. Sous windows renommer la dossier "qui-est-ce-main" par "qui-est-ce".
  a-Retourner dans le dossier qui-est-ce.
5-Executer le fichier "Quiestce-JeuetGenerateur[Windows/Linux].bat" selon OS lorsque vous souhaitez jouer ou creer une grille.
  a-Modifier le chemin du java si vous n'etes pas sur le linux de la fac.

Pour Jouer:
a-Vous pouvez choisir la grille sur laquelle vous voulez jouer en cliquant sur l'engrenage en haut a droite de l'ecran d'acceuil,
 par default "grillesimple" est selectionne.
 (/!\DÃ» a l'utilisation de version anterieur de GLX sur les machines de la fac veuillez imperativement selectionner une grille avant de jouer)
b-Ensuite vous pouvez cliquer sur :
  1 jouer :
   >puis cliquer sur le mode que vous voulez (le seul qui fonctionne est le mode solo).
   >puis cliquer sur le nombre de personnages que vous voulez trouver 1 ou 2.
   >puis cliquer sur "jouer".
  2 sauvegardes :
   >puis sur la sauvegardes que vous souhaitez reprendre (il y a les caracteres "2c" devant les parties avec 2 personnages caches).
   >puis sur "select".

Pour Creer une Grille:
a-Vous pouvez cliquer sur l'icone en forme de grille en haut a gauche.
b-Puis cliquer sur "nouvelle grille" pour creer une nouvelle grille.
 b2-Ou alors choisir une grille a modifier (les grilles finissant par "SAVE" sont des grilles pas non termine ne pouvant etre jouer,
 les autre sans le mot "SAVE" sont jouables).
 b3-Puis cliquer sur "modifier grille" lorsque selectionne.
